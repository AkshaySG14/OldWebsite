            <h1>
                Magnetic Motion Demo
            </h1>
            <p>
                Magnetic Motion Demo is a simple demonstration of physics (mostly momentum and acceleration) created in
                Java. Originally created as a math project, this demo is operated exclusively through the mouse.
                <br/><br/>
                The controls are as follows. A left click will shoot the ball from the launcher.
                A right click will spawn a negative electric charge. This electric charge will deflect the ball.
                An additional right click will change the negative charge into a positive one, which will attract the ball.
                Clicking the charge once more will delete it. The middle mouse button will spawn a bumper, which can interact with
                the charges the same way the ball does. It will also rebound with balls, simulating the act of transfer of momentum.
                <br/><br/>
                The two facets of magnetic attraction and momentum should serve as a fun physics demo. A jar is available for easy
                access to the demo.
            </p>

            <br/>

            <h3>
                How and Why Magnetic Motion Demo Was Made
            </h3>

            <p>
                The Magnetic Motion Demo was originally created as my final project for Calculus AB. However, functionally it
                served as my first foray into the world of Libgdx and the many advantages it offers. Because of the integration
                of the Libgdx framework, this project took me roughly two weeks, starting with my education of how to work with
                Libgdx and ending with the actual implementation of the code. Though a small project, it is a rather cool demonstration
                of physics if I do say so myself, and can be useful as a learning tool for those who have just been introduced to
                Libgdx.
                NOTE: As this was my first project in Libgdx, the coding (especially the methods for rendering) is a bit raw and
                unoptimized. Regardless, it is easily read and comprehensible.
                <br/><br/>
                Language-wise, the Magnetic Motion Demo is completely written in Java, and is built upon the Libgdx Java framework. However,
                Libgdx is mostly used for the rendering of the sprites and the background, and thus is largely unnecessary to
                understand the code of this game. The game can be understood with a beginner's grasp of Java, and can be used as a
                learning tool.
                <br/>
            </p>

<script>
    $(function() {
        // Gets the div that houses the source code.
        const activeSource = $("#active-source");
        // Initialize variable responsible for deciding source code to display.
        var source = 0;

        // Renews the PrismJS script so that the code is properly highlighted.
        function renewPrism() {
            $('script[src="Static/js/prism.js"]').remove();
            $('<script>').attr('src', "Static/js/prism.js").appendTo('head');
        }

        // Script responsible for setting the text of the source code being displayed.
        function setScript() {
            switch (parseInt(source)) {
                case 0:
                    // Sends an async request to load the relevant source code, and then reinitializes the
                    // highlighting script (prism.js).
                    activeSource.load('Static/SourceCode/MagneticMotion/MagneticMotion.html', renewPrism);
                    break;
                case 1:
                    activeSource.load('Static/SourceCode/MagneticMotion/GameScreen.html', renewPrism);
                    break;
                case 2:
                    activeSource.load('Static/SourceCode/MagneticMotion/Map.html', renewPrism);
                    break;
                case 3:
                    activeSource.load('Static/SourceCode/MagneticMotion/MapRenderer.html', renewPrism);
                    break;
                case 4:
                    activeSource.load('Static/SourceCode/MagneticMotion/MainMagnet.html', renewPrism);
                    break;
                case 5:
                    activeSource.load('Static/SourceCode/MagneticMotion/Magnet.html', renewPrism);
                    break;
                case 6:
                    activeSource.load('Static/SourceCode/MagneticMotion/Ball.html', renewPrism);
                    break;
                case 7:
                    activeSource.load('Static/SourceCode/MagneticMotion/Bumper.html', renewPrism);
                    break;
                case 8:
                    activeSource.load('Static/SourceCode/MagneticMotion/Shooter.html', renewPrism);
                    break;
            }
        }

        // Clears all the HTML of the source code div.
        function clearScript() {
            activeSource.html("");
        }

        // When clicking a specific source code to view in the dropdown bar.
        $(".dropdown-item").click(function () {
            $('#dropdownMenuButton').text($(this).text());
            source = $(this).attr('data-val');
            clearScript();
            setScript();
        });
        // Sets initial source code.
        setScript();
    });
</script>
